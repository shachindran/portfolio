Wan and Whiskers

Game Development & OOP Logic

Overview

A high-octane, two-player action game requiring intensive coordination to survive progressively scaling waves of aggressive enemies.

Technical Execution

OOP Architecture: Used this project as a rigorous application of C# and Object-Oriented Programming (OOP) principles. By utilizing inheritance, polymorphism, and encapsulation, I built a highly scalable and modular game architecture.

Complex Game Logic: Engineered systems to manage dynamic multi-character control alongside destructible environmental physics, optimizing the code to ensure responsive gameplay even under heavy processing loads.

Dynamic Scaling: Programmed a tiered endless adventure mode. Enemy behaviors, spawn rates, and environmental hazards evolve dynamically, ensuring the difficulty curve tests both player reflexes and long-term survival strategy.

Core Technologies

Unity Engine C# Object-Oriented Programming (OOP)
